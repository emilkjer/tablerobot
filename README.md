Toy Robot Simulator
==========
By Emil Kjer
Date: 08/10/2012
==========

Usage: ruby run.rb <DATAFILE>
E.g. ruby run.rb data/test_basic.dat

Test files are located under the folder "data". One command per line and the expected format of commands are:
PLACE X,Y,F
MOVE
LEFT
RIGHT
REPORT

Run rspec test by 'rspec spec'.

Enjoy!
